# 功耗计算器

计算嵌入式设备电池续航时间或所需电池容量，支持多种电池类型和工作模式配置。

**功能特性**：
- 多种计算模式：计算续航时间 / 计算所需电池容量
- 多种电池类型：锂电池、一次性锂亚电池、碱性干电池
- 多工作模式：支持添加、删除、编辑多种工作模式
- 图表可视化：柱状图 + 环形图展示能耗分布

**开源地址**：[GitHub](https://github.com/stark1898y/Power-Consumption-Calculator) | [Gitee（国内）](https://gitee.com/stark1898/power-consumption-calculator)
