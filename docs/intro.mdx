# 知识库简介

欢迎来到 **Stark1898y 的个人知识库**！

这是一个专注于嵌入式开发的技术知识库，记录日常学习和工作中的技术笔记、经验总结。

## 知识分类

### 硬件设计

电源、接口、保护器件、常用器件、无线通信等硬件相关知识。

### MCU 开发

- **STM32**：STM32 系列 MCU 开发
- **WCH RISC-V**：CH32V 系列 RISC-V MCU 开发
- **BLE 蓝牙**：CH592、CH584 等 BLE 芯片开发

### 软件开发

框架架构、常用组件、存储方案、通信协议、升级方案等嵌入式软件知识。

### RTOS 系统

RT-Thread、FreeRTOS 等实时操作系统的学习笔记。

### 开发工具

Keil MDK、MounRiver Studio、VS Code 等开发工具的使用技巧。

### Linux 开发

Linux 基础、驱动开发、交叉编译等知识。

### AI & Python

Python 基础、机器学习、AI 工具等学习笔记。

### 学习资源

推荐博主、常用网站、推荐书籍等学习资源汇总。

## 内容来源

- 个人学习笔记
- 工作经验总结
- 优秀博客文章整理
- 官方文档摘录

## 如何使用

1. 使用左侧导航栏浏览不同分类
2. 使用搜索功能查找特定内容
3. 点击链接查看相关资源
