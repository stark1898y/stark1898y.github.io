# 气体浓度换算

可燃气体（甲烷、乙烷、丙烷等）ppm、VOL% 和 LEL 之间的在线换算工具。

**功能特性**：

- 支持 ppm → VOL% → LEL 互转
- 支持多种可燃气体类型
- 支持自定义气体参数

**开源地址**：[GitHub](https://github.com/stark1898y/Gas_VOL_ppm_LEL)
